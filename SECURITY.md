# Security Policy

## Supported Versions

This project is maintained on the default branch (`main`) and accepts security fixes there.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately.

- Open a private GitHub security advisory for this repository

Please include:
- Affected component(s)
- Reproduction steps
- Potential impact
- Suggested mitigation (if available)

## Response Expectations

- Acknowledgement target: within 3 business days
- Initial triage target: within 7 business days
- Remediation timeline: based on severity and exploitability

## Disclosure Guidance

Do **not** open public issues for unpatched vulnerabilities.

Do **not** include live secrets, tokens, or customer data in reports, logs, screenshots, or proofs of concept.
